## 如何使用？

![icon](./icon.png)

安裝 Chrome extension [Excalidraw custom font](https://chrome.google.com/webstore/detail/excalidraw-custom-font/afbeaojffbjckicjpkecknoocdpmgoah)

1. 進入 Excalidraw
2. 輸入 URL 按下 Change font
3. 切換夜覽/白天模式 (觸發字體更新)
